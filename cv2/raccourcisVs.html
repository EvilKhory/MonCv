Les raccourcis VS Code

Utilités

Les raccourcis, (ou shorcut en anglais) sont indispensable pour coder efficacement et se faciliter la vie. Evidement, ces raccourcis fonctionnent sur Windows, mais aussi sur MacOS avec ses equivalents.
Il en existe nativement avec vscode, mais certains raccourcis demande d'installer une extension, nous allons voir ensemble ceux qui necessite une installation et ceux qui sont natif a Vscode.


Recherche globale

Ctrl + Shift + F

Sert a rechercher des mots dans le projet en entier, pratique sur un gros projet.

Selection multiple des occurrences

Ctrl + D

Très utile pour eviter de tout changer mot par mot, permet de selectionner un mot qui se repete afin de tous les modifier

Auto-formatage

Shift + Alt + F

Permet de trier son code afin de le rendre plus lisible et pratique a utiliser

Sauvegarde rapide

Ctrl + S ( Connu de toutes et tous mais il faut le rappeler, c'est comme s'hydrater )
Ctrl + K + F ( Ctrl+K relacher puis F )

Permet de sauvegarder tous les fichiers ouvert en même temps
